## 🧩 What rubui does

rubui is a 3D Rubik’s Cube app for your terminal. You can move the cube by hand or let the app solve it for you. It uses color, motion, and keyboard input to make the cube easy to use on Windows.

This app is made for people who want a simple way to play with a Rubik’s Cube on a PC. You do not need to know how to code.

## 🧠 Manual mode

Manual mode lets you solve the cube step by step.

Use this mode if you want to:

- Practice cube moves
- Learn how each face works
- Try to solve the puzzle on your own
- Explore the cube in 3D

Manual mode keeps the experience close to a real cube, but without holding one in your hands.

## 🤖 Auto mode

Auto mode solves the cube for you.

Use this mode if you want to:

- Watch the cube solve itself
- Test the solver
- Compare solution steps
- See smooth move animations

This mode uses a known solving method so the cube can return to the solved state after a scramble.
